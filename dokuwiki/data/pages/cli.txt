====== Command Line Interface Manual ======

===== Installation =====

<code>
cd cli;
sh ./install.sh
</code>

===== First steps =====

Just run :

<code>
cpm
</code>

Some built in documentation exists within the command interface (available commands and description). Every command/subcommand have a ''--help'' documentation (some are empty).
===== Module Management =====

- Reload the module definitions : (if modifications have been made on .module files)

<code>
cpm reload
</code>

- List all available modules

<code>
cpm module ls
</code>

- Run the module //MODULE_NAME// with optional (but often required) configuration file //CONF_FILE// (hosted on server)
<code>
cpm run MODULE_NAME [CONF_FILE]
</code>

returns a <process id>

===== Run Management =====

<code>
cpm process ls [-a]
</code>
