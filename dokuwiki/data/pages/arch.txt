====== Architecture ======

===== Informations générales =====

AppFM est construit sur un modèle client-serveur. Il est composé d'un kernel (projet /appfm/core), et deux deux interfaces client (projet /appfm/cli et /appfm/web-ui). Un dépot git global contient en sous projet ces différents composants (git@github.limsi.fr:/appfm/base) contenant notamment des scripts d'installation et certaines librairies nécessaire au projet /appfm/core.

===== Kernel (Serveur) =====

- Développé principalement en Scala

- Dépend notamment de : ZMQ, Scala, Java8

- Build tool : SBT

- Nécessite une instance de mongodb

===== Clients =====

**CLI**

- Développé en python

- Dépend notamment de : pyzmq, click

**Web interface**

- Développé principalement en javascript et php5

- Dépend notamment de : zmq

- Build tool (js) : gruntjs

- Nécessite un serveur apache
