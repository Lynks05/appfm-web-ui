====== CPM NLP ======

===== (Quick) Installation =====

1) Get the source

Without GIT : download static archive (with all/maximum dependencies)

With GIT (install git if not : apt-get install git or yum install git, ...)

git clone git@versatile-world.net:~/cpm.git . ; cd cpm ; git clone git@versatile-world.net:~/cpm-cli.git . ; git clone git@versatile-world.net:~/cpm-core.git . ; ...

2)

A. Quick sandbox with vagrant

2.1) install vagrant

2.2) run start.sh

B. Native installation

2.1) check if dependencies are installed and proper configuration file to link them OR use those provided in "lib" (git clone git@versatile-world.net:~/cpm-lib.git . IF with GIT)

2.2) proper configuration of env.sh (edit CPM_HOME in env.sh, ...)  then run source env.sh

2.3) run ./start.sh
